# Datnt114's' dotfiles

## Requirements
- [A Nerd Font](https://www.nerdfonts.com/font-downloads) (it's for the icons)
- [kitty](https://sw.kovidgoyal.net/kitty/) (a fast GPU based terminal emulator)


## Install 
```bash
cd ~
git clone https://github.com/datnt153/dotfiles.git
cd dotfiles 
stow .
```
